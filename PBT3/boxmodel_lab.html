📦 Box Model Lab

Phần 1 — content-box vs border-box

Cả 2 hộp đều có: width: 300px | padding: 20px | border: 5px solid

Mở DevTools → Computed tab → xem box model diagram để đo chiều rộng thực tế.

Hộp 1: content-box

Chiều rộng thực =
300 + 20×2 + 5×2 = 350px

Content: 300px

Padding: 20px mỗi bên

Border: 5px mỗi bên

box-sizing: content-box
Width thuộc tính chỉ là content area
Thực tế = 350px
Hộp 2: border-box

Chiều rộng thực = 300px

Content: 300-20×2-5×2 = 250px

Padding: 20px mỗi bên (trong 300px)

Border: 5px mỗi bên (trong 300px)

box-sizing: border-box
Width bao gồm padding + border
Thực tế = 300px
⚠️ Mở DevTools (F12) → click vào từng hộp → tab "Computed" hoặc "Layout" để thấy box model diagram trực quan. Chụp screenshot!

Phần 2 — Layout 3 cột (container 1000px)

❌ Không dùng border-box (bị vỡ layout)

Sidebar: 250+15×2 = 280px | Content: 500+20×2 = 540px | Ads: 250+15×2 = 280px

Tổng = 280 + 540 + 280 = 1100px — vượt 1000px, layout bị vỡ!

Sidebar

width: 250px

padding: 15px

Thực tế: 280px

Content

width: 500px

padding: 20px

Thực tế: 540px

Ads

width: 250px

padding: 15px

Thực tế: 280px

✅ Dùng border-box (layout đúng)

Với border-box: 250 + 500 + 250 = 1000px — khớp chính xác!

padding: 15px (bên trong)

Content

width: 500px

padding: 20px (bên trong)

Ads

Bonus — Margin Collapse

Box A có margin-bottom: 25px, Box B có margin-top: 40px

Khoảng cách giữa 2 box = max(25, 40) = 40px (KHÔNG PHẢI 65px)

Box A — margin-bottom: 25px
Box B — margin-top: 40px

👆 Mở DevTools để xác nhận khoảng cách thực tế giữa 2 box là 40px
