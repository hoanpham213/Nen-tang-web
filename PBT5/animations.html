Tóm tắt kỹ thuật

  • Card hover: transform: translateY(-8px) + box-shadow tăng, transition: all 0.3s ease
  • Image zoom: overflow: hidden trên .card-image + img transform: scale(1.1) khi hover
  • Button hover: đổi background-color + transform: scale(1.05)
  • Spinner: @keyframes spin { to { transform: rotate(360deg) } } + animation: spin 1s linear infinite
  • Fade-in: @keyframes fadeIn + animation-delay để stagger từng box
